Event Log

What is this? Server Guardian Warboard is a read-only hackathon command view, built with Codex and GPT-5.6, that turns honeypot-style events into an RPG/NORAD defense scene. Battles are narrative renderings; each monster represents redacted bot activity. Confirmed attackers are also filed to a community threat-intelligence feed (AbuseIPDB) by the backend, so defending one small server helps the wider internet. IP addresses are shown as bot aliases, and potentially malicious URLs and commands are shown only in defanged hxxp/[.] form. · Controls: map/NORAD/theme/audio/share toggles; add ?demo=skirmish|dq|srw|fps|fpss|fpsm for deterministic demo scenes. · Devpost testing: no account required, no write actions, and the public viewer is privacy-preserving. · Built for OpenAI Build Week with Codex and GPT-5.6. · Assets: 0x72, LuizMelo, Kenney, Ansimuz and others (CC0) + Pimen free assets; SFX by Juhani Junkala, Kenney, Ogrebane, thebardofblasphemy (CC0); font Galmuri (OFL).

OpenAI Build Week Submission Notes

Track

Apps for Your Life. Managing my own server is a personal hobby, and this turns everyday defense of it into something worth watching without exposing private infrastructure.

How to Test

Open this URL, wait for the live feed, switch between RPG, map, and NORAD views, then try the demo query strings for repeatable judging scenes.

Privacy

Public IPs are replaced by bot aliases, host identity is generalized, and suspicious strings are defanged before display.

Codex + GPT-5.6

Codex and GPT-5.6 were used to extend the viewer, harden display paths, add deterministic QA checks, and prepare this judging surface.

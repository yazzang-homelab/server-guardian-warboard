Definition of Done

데모, 저장소, 3분 미만 공개 YouTube 영상이 비로그인 상태에서 열리고 /feedback 세션 ID를 포함한 Devpost 제출이 완료되어야 최종 완료입니다. 현재 남은 작업은 최신 저장소 변경 공개, YouTube 업로드, 세션 ID 입력, Devpost 제출입니다.

Completion requires a signed-out-accessible demo, repository, public sub-three-minute YouTube video, and a submitted Devpost entry containing the /feedback Session ID. Remaining work is publishing the latest repository changes, YouTube upload, Session ID entry, and Devpost submission.

This guide summarizes preparation work; the official rules remain authoritative.
